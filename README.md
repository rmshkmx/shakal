# Зыбучий Шакал (Shakal) — Material 3 Expressive Android App

Нативное Android-приложение для ухудшения качества фотографий («шакализации») и создания мемов в ультрасовременном дизайне **Material 3 Expressive** (M3E). Приложение полностью переписано с Flutter на **Kotlin** и **Jetpack Compose** для лучшей производительности, нативного вида и плавных анимаций.

---

## 🚀 Основные возможности

1. **Режим «Шакал» (Quality Degradation)**:
   - Сжатие изображения и наложение JPEG-артефактов в реальном времени.
   - Интерактивные слайдеры с тактильным откликом (Haptic Feedback) для регулировки качества и коэффициента уменьшения (downscale).
   - Легкое «пружинное» нажатие (Spring Physics) на контейнер с фото.

2. **Режим «Текст» (Meme Generator)**:
   - Полноценный генератор мемов на основе выбранного фото.
   - Наложение верхнего и нижнего текста легендарным шрифтом **Impact** с черным контуром.
   - Настройка размера текста кнопками увеличения/уменьшения.

3. **Интерактивность и Визуальный Стиль**:
   - Центральная кнопка выбора фото с изменяемой формой (случайная M3E-фигура при каждом запуске приложения) и плавным бесконечным вращением в пустом состоянии.
   - Плавное переключение тем оформления с помощью анимации **«разливания волны» (Circular Reveal Overlay)** прямо от кнопки переключателя.
   - Умная плавающая кнопка сохранения (Save FAB) без лишнего текста, которая плавно анимируется и превращается в крутящуюся 9-угольную фигуру во время сохранения.

---

## 💎 Что нового в последнем обновлении

- **Исправление предпросмотра (Preview Autofit)**:
  - Исправлен баг, из-за которого сильно «зашакаленные» изображения с маленьким физическим разрешением (например, 50x50 пикселей) открывались в предпросмотре крошечными.
  - Теперь предпросмотр автоматически масштабирует и растягивает изображение на весь экран с сохранением исходных пропорций (Aspect Ratio), независимо от степени его сжатия.
- **Интеграция генератора мемов**:
  - Добавлена новая вкладка управления текстом через `HorizontalPager` с красивым переключением страниц в стиле M3E `SegmentedButton`.
- **Тактильный отклик (Haptic Feedback)**:
  - Слайдеры вибрируют при прокрутке на каждом новом шаге.
  - Сохранение изображения также подтверждается короткой тактильной вибрацией.

## 🛠 Сборка и запуск

Для ручной сборки APK-файла выполните следующую команду в корневой папке проекта `shakal_compose`:

```bash
.\gradlew.bat assembleDebug
```

После завершения сборки готовый APK-файл для установки на телефон будет находиться по пути:
`shakal_compose/app/build/outputs/apk/debug/app-debug.apk`
